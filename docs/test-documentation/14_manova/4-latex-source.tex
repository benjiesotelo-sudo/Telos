Tests whether groups differ on a set of outcomes jointly. Read the multivariate p (Pillai's trace is robust) first; if significant, the per-DV follow-up ANOVAs show which individual outcomes drive it. Because you run one ANOVA per outcome, correct those follow-up p-values for multiple comparisons (e.g. Bonferroni: divide alpha by the number of outcomes) before calling each significant. Your significance threshold ($\alpha$) is 0.05.

A MANOVA gave Pillai's V=.29, F(4,114)=4.74, p = .001.
