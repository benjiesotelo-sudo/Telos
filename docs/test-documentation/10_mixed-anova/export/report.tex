Read the Group $\times$ Condition interaction first --- a significant interaction means the groups changed differently across conditions; only then read the main effects. Check sphericity for the within and interaction terms --- if violated, read the corrected F/p. When the group $\times$ condition interaction is significant, the overall condition comparisons in the post-hoc table can mislead --- read each group's line on the profile plot instead. Your significance threshold ($\alpha$) is 0.05.
