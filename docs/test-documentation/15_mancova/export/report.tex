Like MANOVA, but group differences on the set of outcomes are assessed after controlling for one or more covariates. Interpret the univariate follow-ups only if the multivariate p is significant, and adjust them for the number of DVs (e.g. Bonferroni) to control familywise error. Your significance threshold ($\alpha$) is 0.05.

A MANCOVA gave a covariate-adjusted group effect, Pillai's V=.37, F(4,112)=6.30, p $<$ .001.
