The nonparametric counterpart to one-way ANOVA. The H/p tests whether any group tends to have systematically higher or lower values (stochastic dominance) --- read it as a "median difference" only when the groups have similar distribution shapes. If significant, Dunn's post-hoc shows which pairs differ, with adjusted p-values. $\varepsilon$\textsuperscript{2} is the effect size. Your significance threshold ($\alpha$) is 0.05.

A Kruskal-Wallis test gave H(2)=6.56, p = .038, $\varepsilon$\textsuperscript{2}=.11 [.03, 1.00].
