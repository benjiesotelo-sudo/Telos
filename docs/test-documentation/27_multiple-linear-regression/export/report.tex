Each predictor's B is its effect holding the others constant; p and CI assess it. R\textsuperscript{2} is the joint variance explained; VIF flags predictors that overlap too much (multicollinearity). B is in each predictor's own units, so to compare which predictors matter most use the standardized coefficient ($\beta$), not B. Your significance threshold ($\alpha$) is 0.05.

The model explained R\textsuperscript{2}=.75 of the variance, F(5,34)=20.42, p $<$ .001; predictor pre\_score gave B=0.61, p $<$ .001.
