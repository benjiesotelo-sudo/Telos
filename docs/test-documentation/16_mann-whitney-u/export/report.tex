The nonparametric counterpart to the independent t-test, comparing whole distributions via ranks rather than means. A p below alpha means one group's values tend to be systematically higher or lower (stochastic dominance) --- this equals a difference in medians only when the two groups have similar distribution shapes. r gives the effect size. Your significance threshold ($\alpha$) is 0.05.

A Mann-Whitney U test gave U=0, Z=$-$2.88, p = .002, r=$-$1.00 [$-$1.00, $-$1.00].
