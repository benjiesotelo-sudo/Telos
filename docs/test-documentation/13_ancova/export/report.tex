Compares group means after statistically removing the influence of a numeric covariate. First confirm the factor$\times$covariate interaction is non-significant (homogeneity of slopes) --- if it is significant, the adjusted-means interpretation is not valid and a different model is needed. Then read the Factor row's F/p for the adjusted group effect, and the adjusted means for the covariate-controlled group values. Your significance threshold ($\alpha$) is 0.05.

Controlling for the covariate, an ANCOVA gave F(2,56)=8.07, p $<$ .001, partial $\eta$\textsuperscript{2}=.22 [0.07, 1.00].
