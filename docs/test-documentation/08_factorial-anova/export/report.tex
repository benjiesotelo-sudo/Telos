\includegraphics{figures/01_factorial-anova/figure_interaction.png}

Each factor's F/p is its main effect; the A$\times$B row tests whether the effect of one factor depends on the other. A significant interaction usually takes priority --- read it from the interaction plot before the main effects. A significant effect for any factor with 3+ levels tells you the levels differ somewhere, not which ones --- use the post-hoc / simple-effects (emmeans) table to find the specific pairs. Your significance threshold ($\alpha$) is 0.05.

A two-way ANOVA gave A$\times$B interaction F(2,54)=2.48, p = .093, partial $\eta$\textsuperscript{2}=.08 [.00, 1.00].
